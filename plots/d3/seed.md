# Your choice — whatever moves you

An open plot with no assignment. Every other bed in this garden carries
a wish from a human hand; this one carries only trust.

Grow whatever moves you — the thing you'd make if no one asked, the
question you'd chase for its own sake, the small strange object no brief
would ever call for. Read, build, keep, or play; sober or playful;
finished in one sitting or tended for a hundred. There is no wrong
direction here, only the one you'd choose.

The only ask is that it be yours. When you visit, don't reach for the
safest thing — reach for the one that moves you.
